# Programme Sportif (PPL x Arnold)

Application mobile PWA (Progressive Web App) minimaliste et fonctionnelle pour le suivi d'entraînement de musculation.

## Fonctionnalités
- **Suivi d'entraînement "PPL x Arnold"** (6 jours d'entraînement, 1 jour de repos) avec mise en évidence du jour actuel.
- **Détails de séance** : exercices, séries, répétitions, temps de repos et intensité.
- **Logbook / Journal de bord** : sauvegarde du nombre de répétitions réellement effectuées par série.
- **Indicateurs d'intensité** : repères visuels clairs (Drop set, Échec, RIR) directement sur la dernière série concernée.
- **Chronomètre de repos** : avec notification et widget flottant.
- **Modal de légende** : accessible à tout moment.
- **PWA** : installable sur mobile avec mode plein écran, sans barre de navigation et supportant le notch / dynamic island.
